Installing Applications

https://market.webdesk.me

Help

Open Install Folder

  1. Click the WebDesktop power button
  2. Click the yellow "Systems Files" button(This navigates you to a folder)
  3. Click the "App" folder
  4. Go to https://market.webdesk.me and download the zip for the app you wish to install
  5. Extract the zip into the "App" folder
  6. Close and reopen WebDesktop and your app will be there
