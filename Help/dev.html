Help

With WebDesktop you can program the way you choose. You can use vinilla javascript, html, css, node and a mirad of framworks!

  1. Click the WebDesktop power button
  2. Click the yellow "Systems Files" button(This navigates you to a directory)
  3. Click the "App" directory
  4. Create a directory for your app in the "App" directory
  5. Put your app in the new directory you made
  6. Create a file called "wd.json" example {"name": "MyApp", "des": "This is a really fun game!"}
  7. Include a picture for a desktop icon called "ic.png"
